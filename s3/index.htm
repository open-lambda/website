Background

Cloud computing is evolving. All major public cloud providers now support serverless computing:

AWS Lambda
learn more
Google Cloud Functions (Alpha)
learn more
Azure Function
learn more

Serverless computing introduces many new research challenges in the areas of sandboxing, session management, load balancing, and databases.

Unfortunately, all the major implementations of serverless computing platforms are proprietary, so these areas are difficult for academic researchers to explore.

Our goal with OpenLambda is to create an open source implementation of a serverless compute platform, thus enabling this research.
